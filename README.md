# Diabetes-prediction-app

This is a Streamlit-based web application for predicting diabetes using a tuned RandomForest model on the Pima Indians Diabetes dataset.

# Model
- Algorithm: Random Forest (class-weighted)
- Input features: 6 clinical variables
- Output: Diabetes (Yes / No) with probability

# Deployment
The model was trained and deployed using Streamlit Cloud.
